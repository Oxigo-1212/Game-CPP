# TỰA ĐỀ GAME: ZOMBIE SHOOTER
BÀI TẬP LỚN - MÔN LẬP TRÌNH NÂNG CAO
Sinh viên: Nguyễn Phan Việt Hưng - 24021502

# GIỚI THIỆU
Zombie Shooter là một game 2D góc nhìn top-down, nơi người chơi sẽ chống lại các đợt tấn công liên tục của zombie với số lượng ngày càng tăng. Game được phát triển hoàn toàn bằng C++ và SDL2, thể hiện nhiều kỹ thuật lập trình nâng cao như quản lý bộ nhớ, đa luồng, và các thuật toán AI.

# TÍNH NĂNG NỔI BẬT
Hệ thống vũ khí đa dạng: Ba loại vũ khí (Súng lục, Súng trường, Súng shotgun) với đặc tính khác nhau
Thuật toán AI thông minh: Zombie di chuyển theo mô hình Boids (Flocking Behavior) tạo cảm giác thực tế
Thế giới rộng lớn: Sử dụng kỹ thuật Chunk Loading để tối ưu hóa hiệu suất
Hệ thống wave: Độ khó tăng dần theo thời gian, số lượng zombie nhiều hơn và mạnh hơn
Đồ họa đẹp mắt: Animation mượt mà, hiệu ứng bắn súng chân thực
Âm thanh sống động: Hiệu ứng âm thanh cho các hành động như bắn, nạp đạn, zombie chết
Hỗ trợ độ phân giải màn hình linh hoạt: Tự động điều chỉnh theo kích thước cửa sổ
